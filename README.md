# Utilisation de ```bash start_nodes_tmux.sh```

Remarque: Cette version fonctionne avec tous les OS.

Commande:
```bash
  ./start_nodes_tmux.sh -n <nombre de noeuds>
```

Raccourcis clavier pour naviguer dans les fenêtres:
- Changer de +1
```bash
 Control ^ + B, relaché puis N
```
- Changer de -1
```bash
 Control ^ + B, relaché puis P
```
- Aller à l'index i
```bash
 Control ^ + B, relaché puis 0-9
```
